# Portfolio site (inoryy-style)

Static portfolio with: **Home** (bio, interests, education), **Posts**, **Projects**, **Talks**, **Teaching**, **CV**.

## Make it your github.io page

1. **Create a repo** named `ankitmallya.github.io` (if you don’t have it yet).
2. **Copy these files** into the **root** of that repo:
   - `index.html`
   - `style.css`
   - `script.js`
3. **Turn on GitHub Pages:** repo **Settings → Pages →** Source: **Deploy from a branch** → Branch: **main** → Folder: **/ (root)** → Save.
4. The site will be at **https://ankitmallya.github.io**.

## Edit content

- **index.html** — Update biography, interests, education, project cards, talks, teaching, and CV link.
- Add blog post blocks in the **Posts** section (see the commented example in `index.html`).
- Optional: add a CV PDF to the repo and link it in the CV section.

No build step; plain HTML/CSS/JS.
